# 📅 Calendar Feature Guide

## Overview

The Calendar feature helps you visualize your weekly tutoring schedule and track which sessions need reports written.

## How It Works

### Setting Up Your Schedule

1. **Add Recurring Schedules to Students**
   - Go to Students page
   - Edit each student
   - Fill in the "Recurring Schedule" field
   
   **Examples of valid formats:**
   - `Mondays 4pm, Thursdays 6pm`
   - `Mon/Wed/Fri 3pm`
   - `Tuesday 5:30pm`
   - `M/W 2pm, F 4pm`
   - `Monday 10am, Wednesday 2pm, Friday 11am`

2. **The system will automatically:**
   - Parse the schedule
   - Show sessions on the calendar
   - Track which sessions have reports
   - Highlight missing reports

### Using the Calendar

1. **Click "Calendar" in the navigation**

2. **Visual Indicators:**
   - 🟢 **Green border** = Report written
   - 🟡 **Yellow border** = Upcoming session (no report yet)
   - 🔴 **Red border** = Past session missing report
   - **Bold border** = Today

3. **Navigate Weeks:**
   - Use left/right arrows to view different weeks
   - Click "Back to This Week" to jump to current week

4. **Quick Actions:**
   - Click **"Write Report"** on any session to create a report
   - Student and date are pre-filled automatically
   - Just add session details and generate!

## Tips for Best Results

### Schedule Format Tips

The calendar is smart! It understands:
- Day names: Monday, Tuesday, Mon, Tue, M, T, W, etc.
- Times: 4pm, 4:30pm, 16:00, 4:30 PM
- Multiple sessions: Use commas to separate

**What works:**
- ✅ `Mondays 4pm`
- ✅ `Mon/Wed 3:30pm`
- ✅ `M/T/W 2pm, Th/F 5pm`
- ✅ `Monday 10am, Wednesday 2pm`

**What to avoid:**
- ❌ `Every Monday` (just say "Mondays")
- ❌ `4-5pm` (pick one time, like "4pm")

### Workflow Suggestions

**Weekly Routine:**
1. Check calendar on Monday
2. See which sessions from last week need reports (red)
3. Batch write all missing reports
4. Preview upcoming week

**Daily Routine:**
1. After each session, make quick notes
2. Check calendar to see the session
3. Click "Write Report" directly from calendar
4. Student and date auto-filled!

## Calendar Features

### At a Glance

See your entire week:
- All scheduled sessions
- Which students you're seeing
- Report status for each session
- Summary stats (written, upcoming, missing)

### Report Status Tracking

**Green (Completed):**
- Report exists for this date and student
- Click to view/edit the report

**Yellow (Pending):**
- Session is today or in the future
- No report written yet
- Normal - write after the session

**Red (Missing):**
- Session was in the past
- No report written
- Action needed!

### Quick Report Creation

From any calendar session:
1. Click "Write Report" button
2. Opens new report form
3. Student already selected
4. Date already set
5. Just fill in what happened!

## Summary Statistics

Bottom of calendar shows:
- **Reports Written** - Sessions with completed reports
- **Upcoming Sessions** - Future sessions without reports (normal)
- **Missing Reports** - Past sessions without reports (needs attention)

## Frequently Asked Questions

**Q: I don't see my students on the calendar**
A: Make sure to add their recurring schedule when editing the student

**Q: Can I change the schedule format?**
A: The system is flexible! It understands many formats. Just use day names and times.

**Q: What if a student has a one-time schedule change?**
A: The recurring schedule shows typical weeks. You can still manually create reports for any date.

**Q: Can I see past weeks?**
A: Yes! Use the left arrow to navigate to previous weeks.

**Q: What about holidays or canceled sessions?**
A: Simply don't write a report for that date. The calendar shows the *recurring* pattern.

**Q: Can I export the calendar?**
A: Currently, the calendar is for viewing only. Export individual reports from the Reports page.

## Pro Tips

1. **Set schedules for all students** - Makes the calendar most useful

2. **Check calendar daily** - Stay on top of which reports need writing

3. **Use the "Missing Reports" stat** - Quick way to see if you're behind

4. **Write reports from the calendar** - Faster than navigating to Students → New Report

5. **Review last week on Mondays** - Catch any missing reports before they're forgotten
