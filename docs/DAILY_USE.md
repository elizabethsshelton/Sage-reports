# ⚡ Daily Quick Start

## **Fastest Way to Launch:**

### **1. Open Terminal**
Press `Cmd + Space`, type "Terminal", press Enter

### **2. Run This Command:**
```bash
cd "/Users/ellizabethshelton/Desktop/Sage/Sage Reports" && ./launch.sh
```

### **3. That's It!**
Browser opens automatically to http://localhost:3000

---

## **Even Faster: Create a Desktop App**

**One-time setup (2 minutes):**

1. Open **Automator** (Cmd+Space → type "Automator")
2. **New Document** → **Application**
3. Find **"Run Shell Script"** in the left panel
4. Drag it to the right side
5. **Paste this:**
   ```bash
   cd "/Users/ellizabethshelton/Desktop/Sage/Sage Reports"
   ./launch.sh
   ```
6. **File → Save** as **"Sage Reports"**
7. Save to Desktop

**Now:** Just **double-click "Sage Reports"** on your Desktop! 🎉

---

## **To Stop:**

### **Option A: Run Stop Script**
```bash
cd "/Users/ellizabethshelton/Desktop/Sage/Sage Reports"
./stop.sh
```

### **Option B: Just Close the Terminal**
Closing the terminal window stops both servers
